# Lesson 3: Add a Detail Page

This folder contains the lesson 2 project plus a page for one single note.

## What this lesson shows

- Add a route for one note
- Render a second template
- Link from the list page to the detail page
- Understand "read one item" as part of CRUD

## Run

```bash
uv sync
uv run flask --app app run --debug
```

Then open `http://127.0.0.1:5000`.
