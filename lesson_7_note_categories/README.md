# Lesson 7: Add Categories

This folder contains the lesson 6 project plus categories with a second table.

## What this lesson shows

- Create a second table
- Connect notes to categories with `category_id`
- Read related data with SQL `JOIN`
- Work with select fields in HTML forms

## Run

```bash
uv sync
uv run flask --app app run --debug
```

Then open `http://127.0.0.1:5000`.
