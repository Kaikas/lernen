# Lesson 2: Add Editing

This folder contains the base project plus the edit feature.

## What this lesson shows

- Load one note for editing
- Reuse the form for update
- Save changes with an SQL `UPDATE`
- Keep the existing create and delete actions

## Run

```bash
uv sync
uv run flask --app app run --debug
```

Then open `http://127.0.0.1:5000`.
