# Lesson 8: Add a Category Summary

This folder contains the lesson 7 project plus a summary page with SQL aggregation.

## What this lesson shows

- Count rows with SQL `COUNT`
- Group data with `GROUP BY`
- Build a simple reporting page
- Move from CRUD screens to basic analysis

## Run

```bash
uv sync
uv run flask --app app run --debug
```

Then open `http://127.0.0.1:5000`.
