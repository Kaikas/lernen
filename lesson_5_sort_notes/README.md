# Lesson 5: Add Sorting

This folder contains the lesson 4 project plus sorting for the note list.

## What this lesson shows

- Add a sort query string value
- Use a safe whitelist for SQL `ORDER BY`
- Combine search and sorting in one form
- Compare different list views

## Run

```bash
uv sync
uv run flask --app app run --debug
```

Then open `http://127.0.0.1:5000`.
