# Lesson 6: Add Validation

This folder contains the lesson 5 project plus validation and error messages.

## What this lesson shows

- Validate required fields
- Show error messages in the template
- Keep entered form values after invalid input
- Make create and update behave consistently

## Run

```bash
uv sync
uv run flask --app app run --debug
```

Then open `http://127.0.0.1:5000`.
