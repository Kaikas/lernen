# Lesson 4: Add Search

This folder contains the lesson 3 project plus search for the note list.

## What this lesson shows

- Read a query string value
- Search notes with SQL `LIKE`
- Keep the search logic in one helper function
- Understand filtered list views

## Run

```bash
uv sync
uv run flask --app app run --debug
```

Then open `http://127.0.0.1:5000`.
